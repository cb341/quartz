# quartz

Repository for the content published at https://cb341.dev/quartz.

The actual site content lives on the [`content`](https://github.com/cb341/quartz/tree/content) branch.

**Reason:** `main` is the default GitHub branch, and commits there are counted in the global contributions graph. I keep routine publishing work, typo fixes, and similar low-signal edits off `main` so the contributions graph reflects work I consider more meaningful.
