Domain Name System



## Record Types

| Record Type | Description                 | Purpose                       |
| ----------- | --------------------------- | ----------------------------- |
| A Record    | IPv4                        | hostname -> IPv4              |
| AAAA Record | IPv6                        | hostname -> IPv6              |
| CNAME       | Canonical Name / Alias      | multiple DNS -> same IP       |
| TXT         | random attributes / values  |                               |
| MX          | Mail Exchange               | domain name -> email server   |
| ==NS==      | ==Name Server==             |                               |
| ==PTR==         | ==Pointer, reverse DNS lookup== | ==domain name -> canonical name== |
| ==SOA==         |                             |                               |
