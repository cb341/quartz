### Topologies


A bus topology can be thought of as a shared 🚌 line.
One bus route going through multiple bus stops.

```
 Bus stop A                Bus stop C
     🚏                         🚏
|--------------🚌----------------------|
                    🚏
                Bus stop B
```
All devices connect to the same cable.
Each device decides if the packet on the cable is for them or not.
